# CNCF Certification Guide

This maps tools in this repository to CNCF certification exam domains. Use this to focus your study on tools relevant to your target certification.

## CKA (Certified Kubernetes Administrator)

| Exam Domain | Weight | Tools in This Repo |
|---|---|---|
| Storage | 10% | Longhorn |
| Troubleshooting | 30% | Prometheus, Grafana |
| Workloads & Scheduling | 15% | Helm, Kustomize, Karpenter |
| Cluster Architecture | 25% | Kustomize, Crossplane |
| Services & Networking | 20% | Istio |

| Services & Networking | 20% | Istio |
