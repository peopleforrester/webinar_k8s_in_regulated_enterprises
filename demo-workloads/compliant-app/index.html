Compliant Application

This application follows security best practices for regulated environments.

Security controls implemented:

  • Running as non-root user (UID 101)
  • Non-privileged container mode
  • Explicit image version tag (nginx:1.25.4)
  • CPU and memory limits defined
  • Read-only root filesystem with emptyDir volumes
  • All capabilities dropped
  • Seccomp profile: RuntimeDefault
  • Network policy restricting ingress/egress
  • Service account token not mounted

This app passes all Kyverno policies and demonstrates regulatory compliance.
