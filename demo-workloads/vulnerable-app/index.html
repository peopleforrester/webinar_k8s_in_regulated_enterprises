Vulnerable Application

This application is intentionally insecure for demonstration purposes.

Security violations in this deployment:

  • Running as root user (UID 0)
  • Privileged container mode enabled
  • Using mutable :latest image tag
  • No resource limits defined
  • Writable root filesystem
  • Overpermissioned service account with cluster-wide secrets access

This app will be blocked by Kyverno policies after they are applied.
